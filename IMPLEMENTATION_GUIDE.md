## 🎯 5 Major Improvements (5 Cải Tiến Chính)

### ⚠️ IMPROVEMENT #1: Enhanced Mistakes — Self-test + Case Study + Detection
**Nâng cấp Sai Lầm từ "danh sách" → "framework chẩn đoán"**

Mỗi sai lầm giờ có **6 lớp**:
1. **Định nghĩa** — 1 câu sắc gọn
2. **Self-test** — 3 câu hỏi yes/no tự chẩn đoán
3. **Case Study** — Trade thực tế với lý do + hậu quả
4. **Cách làm sai** — Hành vi cụ thể
5. **Cách làm đúng** — Quy tắc thực thi
6. **Detection Rule** — Logic auto-detect từ journal

**7 Mistakes được nâng cấp:**
- FOMO (Đuổi giá)
- Không cắt lỗ
- Giao dịch quá nhiều
- Đánh ngược thị trường
- Vào vị thế quá lớn
- Bán non cổ phiếu mạnh
- Không ghi nhật ký

**Cách sử dụng:**
1. Vào tab **⚠️ Sai Lầm**
2. Click bất kỳ lỗi nào → xem 6 lớp nội dung
3. Làm **Self-test** (3 câu hỏi) → auto tính Risk Level
4. Xem **Case Study** thực tế
5. Đọc **Hành động Sửa** để tránh lặp lại

### 💪 IMPROVEMENT #5: Discipline Modules → Real Features
**Mô tả "nên làm" → Tính năng thực sự hoạt động**

**8 Modules bây giờ là công cụ thực thi:**

#### 1. **Pre-Trade Gate** (Khoá lệnh nếu chưa đủ điều kiện)
- **10 tiêu chí checklist** trước khi bấm Save
- Disable nút Save nếu score < 60%
- Mỗi tiêu chí có weight khác nhau
```
VD: Entry khi score ≥ B (60%)
Grade A+ (90-100%) → Vào bình thường
Grade A  (75-89%)  → Vào được
Grade B  (60-74%)  → Vào được (ngưỡng tối thiểu)
Grade C  (<60%)    → CHẶN — cần tick thêm tiêu chí
```

#### 2. **Trade Quality Score** (Auto chấm A+/A/B/C/D)
- Tính từ: % checklist + R:R + market regime
- Hiển thị ngay khi tick checklist
- Color-coded grade

#### 3. **Risk Guardrail** (Cảnh báo khi vượt ngưỡng)
- Max risk/lệnh: **≤2% NAV**
- Max risk tổng: **≤6% NAV**
- Max daily drawdown: **≤8% NAV**
- Toast warning khi vượt → phải xác nhận 2 lần

#### 4. **Loss Autopsy** (Mổ xẻ lệnh thua)
- Modal **bắt buộc** khi đánh dấu LOSS
- Chọn 1 nguyên nhân từ 7 mistakes
- Ghi bài học tối thiểu 20 ký tự
- Lesson được lưu vào journal entry

#### 5. **Mood Check** (Pre-Trade Mood)
- Modal xuất hiện khi click "mở lệnh"
- 4 trạng thái: Calm / Fomo / Fear / Revenge
- Risk multiplier được áp dụng tự động
- Revenge mood chặn lệnh nếu thua ≥2 hôm nay

#### 6. **Market Regime Filter** (Daily xác định xu hướng)
- Modal xuất hiện 1 lần mỗi sáng
- 3 lựa chọn: ↗ Uptrend / ↔ Sideway / ↘ Downtrend
- Badge hiển thị trên Dashboard
- Ảnh hưởng setup recommendation

#### 7. **Mistake Auto-Detect** (Real-time scan)
- Scan journal tự động khi thêm lệnh
- 7 detection rules:
  - Loss streak ≥3 trong 24h → FOMO warning
  - Trades/day ≥5 → Overtrade warning
  - Entries không note → No journal warning
  - Risk > 2% → Oversize warning
- Banner cảnh báo trên Dashboard

#### 8. **Weekly Review Auto** (Tự động tổng kết)
- Cuối tuần: PnL, Winrate, Lỗi lặp lại
- Setup tốt nhất (theo PnL)
- Lỗi phổ biến tuần này
- Streak hôm nay (breathwork, principles)

**Cách sử dụng:**
1. **Trước khi vào lệnh:**
   - Click "+ ENTRY" → Chọn Mood → Tick checklist → Save (bị chặn nếu < Grade B)
2. **Sau khi lệnh LOSS:**
   - Chọn status LOSS → Modal Autopsy → Chọn nguyên nhân → Ghi bài học
3. **Hàng ngày:**
   - Dashboard tự check Market Regime lúc 8 sáng
   - Xem Mistake Alerts
   - Thực hiện Daily Principle
4. **Hàng tuần:**
   - Weekly Review auto-generate trên Dashboard

## 🚀 How to Use (Hướng Dẫn Chi Tiết)

### **On First Login:**
1. Choose **Market Regime** (↗ / ↔ / ↘)
2. Enter **Initial Capital** in Settings
3. Add pairs (XAUUSD, EURUSD, etc.)

### **Create New Entry:**
1. Click **📓 Journal → + ENTRY**
2. **Mood Check** modal appears → Select mood
3. Fill: Date, Pair, Direction, Session, Strategy
4. **Pre-Trade Checklist** → Tick ≥6 criteria for Grade B
5. See Quality Score update in real-time
6. Add chart image
7. Click **💾 Lưu** (Save button only active if Grade B+)

### **Mark Trade as LOSS:**
1. Click status dropdown → LOSS
2. **Loss Autopsy** modal → Choose mistake reason
3. Type lesson (≥20 chars) to prevent repeat
4. Confirm

### **Review Mistakes:**
1. Click **⚠️ Sai Lầm** tab
2. Click any mistake → See 6 layers:
   - Case study
   - Self-test (3 questions)
   - Wrong vs Right
   - Detection rule
3. Dashboard shows **Mistake Radar** alerts

### **Learn Patterns:**
1. **📚 Wiki Setup** → Search L1-01 to L4-04
2. Each pattern: Definition + Setup diagram + R:R + Winrate
3. Use in **Analysis** tab to compare with your trade

### **Daily Principle:**
1. Dashboard → Xem **Daily Principle of the Day**
2. Click "TÔI ĐÃ THỰC HIỆN" → Streak increases
3. **💎 Nguyên Tắc** tab → Filter by category → 30 full cards

### **Psychology:**
1. **🧠 Tâm Lý** tab → Breathing Timer 4-7-8
2. 4s breathe in / 7s hold / 8s exhale (5 cycles)
3. Mood auto-tagged to each entry

## 🔧 Firebase Setup (Quan Trọng!)

**Must replace Firebase config in firebase.js:**
```javascript
const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_PROJECT.firebaseapp.com",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_PROJECT.appspot.com",
    messagingSenderId: "YOUR_SENDER_ID",
    appId: "YOUR_APP_ID"
};
```

**Firestore Rules (Allow read/write):**
```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth.uid == userId;
    }
    match /system/{doc=**} {
      allow read: if true;
      allow write: if exists(/databases/$(database)/documents/users/$(request.auth.uid));
    }
  }
}
```
